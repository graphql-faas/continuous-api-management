句型整理

  • It follows that ...

句型核心意思表达一种逻辑结果,是由某事推断出的一种结果.所以,它可以翻译为:因此,由此可知,等. 例: If a = b and b = c ,it follows that a = c. 如果a = b,且b = c,因此a = c.

  • it’s not a big surprise that ...

that后面接从句,表示什么并不奇怪.
