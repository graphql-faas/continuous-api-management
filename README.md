continuous-api-management chinese  docs

## how to build

* use circleci

```code
yarn publish-gh-pages
```

* local build with docker

```code
docker-compose build
```

## 启动:
* 安装依赖包(进入website目录)
`npm install`

* 运行
`npm start`
